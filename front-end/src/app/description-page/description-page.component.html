Controlled Language Definition

check_circle Use clear and specific terms when referring to ontology entities.
For instance, when mentioning a card in the ontology, use phrases like "card" or "title".

check_circle If involving multiple entities, describe their relationships explicitly.
Example: "Create a template for an card with an image and an input contained in an card content."

check_circle Use the ontology's hierarchical structure in descriptions to minimize ambiguity.
Example: "Create the code for a card which contains a header with a title and a subtitle and also a card content with a paragraph."

check_circle Use verbs like "create", "contains", "include" or other synonyms when expressing relationships or containment.
Example: "Create a template for an card containing an image and an input."
